**What Anthropic shipped:** A multi-agent cybersecurity harness (Claude Mythos Preview) that maps a target codebase, spins up scanning sub-agents, triages findings, and writes reports. A "threat model builder" prioritises work by attack surface before scheduling any sub-agent — rank candidates first, then schedule, rather than using a blind single-item waterfall.

**CSIS mapping:** `csis/curiosity.py` already implements a waterfall (rollback → gap → seed), but items within each tier have an opaque `priority: int` with no normalised score. Adding `score: float` (set at construction: 1.0 / 0.6 / 0.2 / 0.0) and `Curiosity.batch(n, hierarchy)` (sorts n consumed items by score desc) makes the ranking explicit and testable, enabling the Coordinator to reason across a candidate landscape rather than committing to the top of the waterfall blindly.
